# Slidev 3lectrify Theme

A custom Slidev theme aligned with the 3lectrify design system.

## Features

- Custom 3lectrify branding and colors
- Animated components (stagger, flow, pulse glow effects)
- Multiple layout options (cover, center, two-cols-custom, image-right)
- Atomic design principles
- Responsive and modern

## Usage

### Development

```bash
npm install
npm run dev:awesome
```

### Build for Production

```bash
npm run build
```

This will generate a static site in the `dist/` folder ready for deployment.

### Export to PDF

```bash
npm run export
```

## Presentations

- `project-awesome.md` - The "Project Awesome" presentation showcasing the 3lectrify digital transformation journey

## Theme Structure

- `/layouts` - Custom slide layouts
- `/components` - Reusable Vue components
- `/styles` - CSS including design system variables and animations
- `/public` - Static assets (images, icons)

This presentation can be deployed to any static hosting service like:
